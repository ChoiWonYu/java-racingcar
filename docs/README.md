# 기능 구현 목록
<hr/>

## View
게임 필요 사항 입력과 결과 출력에 대한 책임 부여
- [X] 자동차의 이름을 입력받는다.
- [X] 이동 횟수를 입력받는다.
- [ ] 최종 우승자를 출력한다.

## Game 
게임의 진행 주관의 책임 부여
- [ ] 이동 횟수만큼 Car를 전진 혹은 정지시킨다.
  - [ ] 랜덤하게 이동 여부를 결정한다.
- [ ] 게임이 끝나고 우승한 Car를 선정한다.

## Car
이동에 대한 책임 부여
- [ ] 명령대로 움직이거나 정지한다.
- [ ] 본인의 이동 거리를 출력한다.
